## 转发原神抽奖

事情的起因是主播一个同学在群里说芙宁娜现在狗都不看了，主播非常生气但又怼不过他，于是主播晚上回到家熬夜把芙宁娜IP价值算出来了，因为主播只有一个手机还要用来收集各种资料，为了方便主播甚至采用用笔算这种原始方法，最后废掉了两个草稿本三支笔主播成功把芙宁娜IP算出来了，最后算出来长期IP价值大概是560亿元，短期IP价值大约有300亿元，大概100个主播被卖了也值不了这么多，同时第二天主播又不小心把鸣潮IP价值也算出来了，跟芙宁娜差不多也是大约500亿，之后一个月，主播又手滑把原神其他角色IP价值也熬夜算出来，散步、雷电将军、钟离长期IP价值都突破300亿，其他角色IP价值基本都在5亿以上，我把这些发给我同学之后他直接变成哑巴了。
家人们记住了，下次谁在瞧不起你推，你就不小心熬夜把你推IP价值算出来，把那个人吓死。
