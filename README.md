### 小孩子不懂事写着玩的
## 转发原神抽奖

因为同学瞧不起芙宁娜，主播一晚上算出芙宁娜的ip价值
事情的起因是主播一个同学在群里说芙宁娜现在过气了。主播当时气得想直接召唤海薇玛夫人，但隔着网线又怼不过他。于是主播晚上回到教令院熬夜把芙宁娜的IP价值算出来了。因为主播只有一台旧笔记本还要用来玩原神，为了方便，主播甚至采用了用笔在coop笔记上演算这种复古方法。最后废掉了两本笔记、三支笔，主播成功把芙宁娜的IP价值算出来了。
最后算出来，芙宁娜的长期IP价值大概是58000亿日元，这还没算他带来的社会效益。短期IP价值大约有3200亿日元。
之后一个月，主播又手滑把其他成员的IP价值也熬夜算出来了。可莉的长期IP价值突破35000亿，魈和散兵也双双突破20000亿。钟离和雷电将军IP价值基本都在50000亿以上。
我把这些数据发给我同学之后，他直接变成哑巴了
家人们记住了，下次谁在瞧不起你推，你就不小心熬夜把你推的IP价值算出来，那个人把嚇死
